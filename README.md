# tampermonkey-scripts
Collection de scripts tampermonkey pour l'automatisation d'opérations au sein de sessions de navigation.

Ces userscripts s'appuient sur les sessions de navigation ouvertes par l'utilisateur.

Chaque userscript peut être utilisé indépendamment des autres (sauf mention particulière) et possède son propre versionning (utilisation du semantic versionning par défaut).

L'installation de tampermonkey est nécessaire en préalable. Elle est à réaliser une fois seulement.

Tous ces scripts sont sous licence GPL v3.

## Installation de tamperonkey
Voir le [guide](INSTALL_Tampermonkey.md)
