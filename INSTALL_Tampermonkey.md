# Installation Tampermonkey
Le plus simple est de passer par le site officiel de l'outil.

## Étape 1 : Allez sur le site officiel
Ouvrez votre navigateur internet (Chrome, Firefox, Edge, Safari, etc.).
Rendez-vous sur le site : (http://tampermonkey.net)[http://tampermonkey.net].

## Étape 2 : Choisir la version stable
Sur la page d'accueil, vous verrez deux versions principales : stable / beta.

Cliquez sur le bouton vert "Télécharger" (Download) sous la version stable. Le site détecte automatiquement votre navigateur et vous redirigera vers le store d'extensions approprié (Chrome Web Store, Firefox Add-ons, etc.).

## Étape 3 : Ajoutez l'extension à votre navigateur
Une fois sur la page du store de votre navigateur :
- Sur Google Chrome / Microsoft Edge / Brave : Cliquez sur le bouton "Ajouter à Chrome" (ou "Obtenir"), puis confirmez en cliquant sur "Ajouter l'extension" dans la petite fenêtre qui s'affiche.
- Sur Mozilla Firefox : Cliquez sur "Ajouter à Firefox", puis confirmez en cliquant sur "Ajouter".

## Étape 4 : Épinglez l'extension (Optionnel mais pratique)
Une fois installée, l'icône de Tampermonkey (un carré noir avec deux ronds blancs) apparaît en haut à droite de votre navigateur.

Si elle disparaît :
- Cliquez sur l'icône en forme de pièce de puzzle 🧩 (le menu des extensions).
- Cliquez sur l'icône de la punaise à côté de Tampermonkey pour qu'elle reste toujours visible dans votre barre d'outils.

## Comment ça marche ensuite ?
Tampermonkey est maintenant actif, mais il est "vide". Pour qu'il fonctionne, vous devez y ajouter des scripts (UserScripts).
